# 🔄 Machine Learning Lifecycle

Machine Learning follows a structured process from collecting data to deploying a trained model.

---

## 1. Data Collection

Collect relevant data from different sources.

Examples:
- CSV files
- Databases
- APIs
- Sensors

---

## 2. Data Cleaning

Remove missing values, duplicate records, and incorrect data.

---

## 3. Feature Engineering

Select or create useful features that improve model performance.

---

## 4. Model Selection

Choose the appropriate Machine Learning algorithm.

Examples:
- Linear Regression
- Decision Tree
- Random Forest

---

## 5. Model Training

Train the model using training data.

---

## 6. Model Evaluation

Evaluate the model using metrics like:

- Accuracy
- Precision
- Recall
- F1 Score

---

## 7. Deployment

Deploy the trained model for real-world use.

Examples:
- Web Application
- Mobile Application
- Cloud Deployment

---

## 8. Monitoring

Continuously monitor the model and retrain it when necessary.

---
